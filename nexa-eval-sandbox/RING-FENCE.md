# RING-FENCE — read before you touch anything

This is a **synthetic-only** sandbox. These rules are non-negotiable and are both
a security requirement and a client-confidentiality obligation.

**You will NEVER:**

- Use any real client prompts, real client telemetry, real device data, or real
  credentials.
- Point the harness at any real / production Copilot, network, or device.
- Put a real client's name, IP, or credential into any file, prompt, commit
  message, or external tool (including your own Claude / Claude Code).

**You will ONLY:**

- Work against the SynthBank estate and the synthetic prompt set in this package.
- Run the mock Copilot, or a Copilot instance Pramod explicitly gives you that is
  itself running on synthetic data.

**Everything here is fake by design.** SynthBank is an invented bank. Its CIs,
services, hostnames (`*.synthbank.local`), and metrics are generated by
`synthbank/generate.py`. None of it corresponds to any real client.

If you are ever unsure whether something is synthetic or real: **stop and ask
Pramod.** Flag it — don't guess.

Using your own Claude / Claude Code on this sandbox is encouraged. Sharing
synthetic data with it is fine. Sharing anything real is not.
